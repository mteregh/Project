##Project Description

EventHub is a web plataform designed for university communities to create, discover and participate in events such as talks, workshops, sports activities, and social getherings.
Users can register for events, join waiting lists when events are full, and leave reviews after attending.The system also allows organizers to manage events and administrators to moderate the platform.

This repository contains all the required deliverables for Assignment 1:
- index.html: The static HTML Landing Page serving as the public-facing homepage.
- style.css: Custom stylesheet applied to the Landing Page (complementing Bootstrap)
- User_Stories.pdf: Document with the complete set of user stories, defining the platform's main functionalities, roles and event lifecycle.
- Relational_Diagram.jpeg: The relational diagram defining the database structure, entities, attributes and relationships.
